1 Install Termux
Install Termux from F-Droid or GitHub, then run these commands:
2 Install Python & Dependencies 
pkg update && pkg upgrade -y
pkg install python git -y
pip install aiohttp rich click pyyaml
3 Download Model (optional)
For local offline AI, download TinyLlama (~461MB):
pip install ctransformers huggingface-hub
python -c "from huggingface_hub import hf_hub_download; hf_hub_download(repo_id='TheBloke/TinyLlama-1.1B-Chat-v1.0-GGUF', filename='tinyllama-1.1b-chat-v1.0.Q2_K.gguf')"
4 Start Backend Server 
git clone https://github.com/pop31-ai/ai-orchestrator.git
cd ai-orchestrator
python -m ai_orchestrator serve --host 0.0.0.0 --port 8080
